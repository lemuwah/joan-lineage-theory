# Primary Sources Gallery

This section contains the core documentary evidence for Joan / Joane / Anashuecot and the Greene family.  
Every document is presented with:

- Full transcription  
- Context  
- What the document actually says

- Evidence status (🟢 PROVEN, 🟡 PROBABLE, ⚪ UNVERIFIED, ⚫ NULL)
